Описание программы Maze

Здесь вы найдёте краткое описание работы программы Maze и её функционала.

Maze - это программа для генерирования лабиринтов по алгоритму Эллера и нахождению пути между двумя заданными точками. Графический интерфейс реализован с использованием UI framework Avalonia. Сгенерированные лабириты возможно сохранить в файл и загрузить из файла для дальнейшей работы. Лабиринт имеет ограничение - 50х50.
